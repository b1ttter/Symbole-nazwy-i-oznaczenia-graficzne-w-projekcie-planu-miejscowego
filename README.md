# Symbole-nazwy-i-oznaczenia-graficzne-w-projekcie-planu-miejscowego
Repozytorium zawiera zestaw stylów graficznych obejmujących symbole i oznaczenia przeznaczenia terenów stosowane w projektach miejscowych planów zagospodarowania przestrzennego. Style opracowano zgodnie z obowiązującym rozporządzeniem i przeznaczone są do wykorzystania w systemach GIS.
